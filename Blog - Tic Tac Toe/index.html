Tic-Tac-Toe Game Using JavaScript

Introduction

Tic-Tac-Toe is a classic game that has been enjoyed by people of all ages for generations. It is a simple game that requires two players to take turns marking Xs and Os on a 3x3 grid, with the objective of getting three in a row. The game is easy to learn but can become quite competitive once players start to master their strategies.

In this blog post, we will go through the steps of building a Tic-Tac-Toe game in JavaScript, using HTML and CSS to style the game board.

Step 1: Create the HTML Structure

The first step is to set up the HTML structure for our game. We'll start with a form to input player names and a button to start the game.

Some Basics of HTML code, we used:

<!DOCTYPE html> - This declaration tells the browser what type of HTML we're using.

<html lang="en"> - This is the opening tag for the HTML document, with a language attribute set to "en".

<head> - This is where we put meta information about the page, such as the title, links to stylesheets, and scripts.

<title> - This page title will appear in the browser tab.

<link rel="stylesheet" href="style.css"> - This is a link to the external stylesheet that we will use to style the game.

<body> - This is where we put the visible content of the page.

<script src="main.js"></script> - This is a script tag that links to the main.js file where we will write our JavaScript code.

Step 3: Add Interactivity with JavaScript

Now that we have created the HTML structure and styled the game board with CSS, let's add some interactivity to the game using JavaScript.

JS Code

let player1NameInGame; let player2NameInGame; let timeLeft = 60; let timerInterval; let btnRef = document.querySelectorAll(".boardCell"); let msgRef = document.getElementById("message"); let popupRef = document.querySelector(".popup"); let newgameBtn = document.querySelector(".popup__restart-btn"); let restartBtn = document.querySelector(".game-restart-btn"); let xTurn = true; let count = 0; function startButton() { let playerName1 = document.getElementById("player1").value; timeLeft--; if (timeLeft < 0) { clearInterval(timerInterval); alert("Time's up Game Over!"); newGame(); } }, 1000); } startTimer(); let winningPattern = [ [0, 1, 2], [0, 3, 6], [2, 5, 8], [6, 7, 8], [3, 4, 5], [1, 4, 7], [0, 4, 8], [2, 4, 6], ]; const disableButtons = () => { btnRef.forEach((element) => (element.disabled = true)); popupRef.classList.remove("hide"); }; const enableButtons = () => { btnRef.forEach((element) => { element.innerText = ""; Global Variables:

player1NameInGame and player2NameInGame: These variables store the names of the players obtained from the URL parameters.
timeLeft: Initialized to 60 seconds, represents the countdown timer for the game.
timerInterval: Stores the interval ID for the timer.
btnRef: Contains references to all the board cells (buttons) on the game board.
msgRef: Refers to the element with the ID message, presumably used to display game messages.
popupRef: Refers to the popup element used to display game outcomes.
newgameBtn: Refers to the "Restart" button inside the popup.
restartBtn: Refers to the "Reset" button outside the popup.
xTurn: Indicates whose turn it is, initialized to true for player 1's turn.
count: Keeps track of the number of moves played.

startButton() Function:

Retrieves player names from input fields.
Checks if both player names are provided.
If names are provided, redirects to the game page with player names encoded in the URL.

startTicTacToe() Function:

Parses player names from the URL parameters.
Sets the player names in the respective containers on the game page.

startTimer() Function:

Initiates a countdown timer.
Updates the timer display every second.
Stops the timer and triggers an alert when time runs out.

winningPattern Array:

Contains arrays representing the winning patterns on the Tic Tac Toe board.

Helper Functions:

disableButtons(): Disables all buttons on the game board and shows the popup.
enableButtons(): Enables all buttons on the game board and hides the popup.
winFunction(): Displays a message with the winning player's name.
drawFunction(): Displays a message indicating a draw.
newGame(): Resets the game by clearing the board, resetting the timer, enabling buttons, and starting a new timer.
exitGame(): Redirects to the index page.

winChecker() Function:

Checks for winning patterns on the board.
If a winning pattern is found, calls winFunction() with the winning symbol.

Button Event Listeners:

Iterates over each button on the board.
Adds a click event listener to each button.
Updates the button's appearance and text based on whose turn it is.
Increments the move count and checks for a win or draw after each move.
